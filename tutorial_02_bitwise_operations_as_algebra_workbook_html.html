Part A: Warm-up vocabulary

A1. XOR outputs 1 when two bits are different and 0 when they are the same.

A2. AND outputs 1 only when both input bits are 1.

A3. A bit mask is a bit pattern used to select, clear, force, or test certain bit positions.

A4. A shift moves bits and fills empty positions with 0, often losing edge bits. A rotation moves bits but wraps edge bits around to the other side.

Part G: Mixed cryptography-flavored practice

G1. 0x57 XOR 0x83 = 0xD4.

\[ 01010111_2 \oplus 10000011_2 = 11010100_2 \]

G2. 0xD4 XOR 0x83 = 0x57.

\[ 11010100_2 \oplus 10000011_2 = 01010111_2 \]

G3 sample answer. XOR is useful because it is reversible when the key is known. AND masking can clear bits, destroying information. If information is destroyed in a direct encryption path, decryption cannot recover the original value from the result alone.
