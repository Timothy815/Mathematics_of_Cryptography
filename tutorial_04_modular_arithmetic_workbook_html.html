Part A: Warm-up vocabulary

A1. The modulus is the wraparound size, like the number of positions on a clock.

A2. \(a\bmod n\) asks for the remainder after dividing \(a\) by \(n\).

A3. \(a\equiv b\pmod n\) means \(a\) and \(b\) have the same remainder modulo \(n\). Technically, \(n\mid(a-b)\).

A4. There are \(n\) residues. Counting from 0 through \(n-1\) gives exactly \(n\) positions, and \(n\) itself wraps back to 0.

Part H: Challenge problems

H1 sample answer. \(12\equiv0\pmod{12}\) because 12 leaves remainder 0 when divided by 12. But the standard residues modulo 12 are 0 through 11, so 12 wraps back to 0 instead of appearing as its own standard residue.

H2 sample answer. \(-1\equiv25\pmod{26}\) because adding 26 to \(-1\) gives 25. Also, \(-1-25=-26\), and \(-26\) is divisible by 26.

H3 sample answer. An unsigned byte has 256 possible values, 0 through 255. If arithmetic overflows past 255 and wraps around, it behaves like arithmetic modulo 256 because values are reduced back into the range 0 through 255.
