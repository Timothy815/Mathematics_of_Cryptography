The Algorithm

  1. Start with \((a, b)\), \(a \ge b\).
  2. Divide: \(a = qb + r\).
  3. If \(r = 0\), then \(\gcd = b\).
  4. Otherwise replace \((a,b)\) with \((b,r)\) and repeat.

Worked Example

StepDivision\(r\)
1\(48=2\cdot18+12\)12
2\(18=1\cdot12+6\)6
3\(12=2\cdot6+0\)0

\(\gcd(48,18)=6\)

Common Mistakes

Wrong: stop at the 0 remainder and say gcd = 0.

Right: the gcd is the last nonzero remainder.

Wrong: replace with \((a,r)\).

Right: replace with \((b,r)\) — the old divisor becomes the new larger number.

Inverse Existence

\(a^{-1} \pmod{n}\) exists when and only when \(\gcd(a,n)=1\).
