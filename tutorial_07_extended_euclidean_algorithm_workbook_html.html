Bézout's Identity

For any \(a, n\) with \(\gcd(a,n)=1\), there exist integers \(s, t\) such that:

\(1 = s\cdot a + t\cdot n\)

Then \(a^{-1}\equiv s\pmod n\). (The coefficient of \(a\), not \(n\!)\)

Worked Example — 15⁻¹ mod 26

\(26=1\cdot15+11\)

\(15=1\cdot11+4\)

\(11=2\cdot4+3\)

\(4=1\cdot3+1\)

Back-sub: \(1=7\cdot15-4\cdot26\)

\(15^{-1}\equiv 7\pmod{26}\)

Negative Coefficient

If back-substitution gives a negative \(s\), add \(n\):

\(s+n\equiv s\pmod n\)

Example: \(s=-6,\;n=151\)

\(-6+151=145\), so \(25^{-1}\equiv145\pmod{151}\)
