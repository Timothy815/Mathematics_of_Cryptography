How to use this workbook

This workbook is about representation, not multiplication. Your main job is to read each bit as a coefficient. A 1 bit includes the matching power of \(x\); a 0 bit leaves it out.

Quick reference

Bit position map
Position76543210
Term\(x^7\)\(x^6\)\(x^5\)\(x^4\)\(x^3\)\(x^2\)\(x\)\(1\)
Core representation
\[ a_7a_6a_5a_4a_3a_2a_1a_0 \longleftrightarrow a_7x^7+a_6x^6+a_5x^5+a_4x^4+a_3x^3+a_2x^2+a_1x+a_0 \]

A coefficient \(a_k\) is either 0 or 1. If it is 1, include \(x^k\). If it is 0, omit \(x^k\).

Part G: Concept checks

G1 sample answer. As an integer, 01010111 uses powers of 2 and equals 87. As a polynomial, the same bits are treated as coefficients of powers of \(x\), giving \(x^6+x^4+x^2+x+1\).

G2 sample answer. Bit position 0 corresponds to \(x^0\), and \(x^0=1\), so we write the constant term as 1.

G3 sample answer. Multiplication in \(\mathrm{GF}(2^8)\) requires more ideas: polynomial multiplication, coefficient arithmetic modulo 2, and polynomial reduction. This lesson only builds the representation bridge.
