Byte ↔ Polynomial

Each byte is 8 bits. Bit \(k\) (counting from 0 at the right) corresponds to the term \(x^k\).

If bit \(k\) is 1, the term \(x^k\) is present; if 0, it is absent.

Example: 0x57 = 0101 0111
\(x^6+x^4+x^2+x+1\)

Polynomial Degree

The degree is the position of the highest set bit (leftmost 1).

HexBinaryDegree
0x801000 00007
0x570101 01116
0x0F0000 11113
0x010000 00010

Term Count = Popcount

The number of terms equals the number of 1-bits in the byte.

HexBinaryTerms
0xFF1111 11118
0x570101 01115
0x801000 00001

XOR = Polynomial Addition mod 2

Adding two byte-polynomials in \(\text{GF}(2)[x]\) is the same as XOR.

Each coefficient is added mod 2: \(1+1=0\), \(1+0=1\), \(0+0=0\).

Example: 0x57 XOR 0x83

0101 0111 XOR 1000 0011 = 1101 0100 = 0xD4
