Term Cancellation Rule

If a term appears in both polynomials → coefficient 1+1 = 0 → cancels.

If a term appears in one polynomial → coefficient 1+0 = 1 → survives.

No carrying between bit positions.

Worked Example

0x57 = 0101 0111 = \(x^6+x^4+x^2+x+1\)

0x83 = 1000 0011 = \(x^7+x+1\)

0x57 ⊕ 0x83 = 1101 0100 = 0xD4

The \(x\) and constant terms cancel; the rest survive.

Key Properties

  • \(A + A = 0\) for every byte — self-inverse
  • \(A + 0 = A\) — zero is the identity
  • Subtraction = addition in GF(2)
  • Degree = position of the highest 1-bit in the XOR result
