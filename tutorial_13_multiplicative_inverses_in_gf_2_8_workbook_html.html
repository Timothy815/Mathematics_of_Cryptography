How to use this workbook

Remember that an inverse is checked by multiplication in \(\mathrm{GF}(2^8)\), not ordinary integer multiplication. A byte \(b\) is the inverse of byte \(a\) when their finite-field product is 0x01.

Quick reference

Inverse definition
\[ a\cdot a^{-1}=1 \] \[ a(x)b(x)\equiv1\pmod{x^8+x^4+x^3+x+1} \]

In byte form, the identity value is 0x01.

Important boundary
  • 0x00 has no multiplicative inverse.
  • Every nonzero byte has an inverse in AES's \(\mathrm{GF}(2^8)\).
  • The AES S-box uses the inverse step, then an affine transformation.

Note for self-study

The central test is simple: finite-field product equals 0x01. The hard part is remembering that the multiplication must be \(\mathrm{GF}(2^8)\) multiplication, not ordinary multiplication.

Part A: Warm-up vocabulary

A1. A multiplicative inverse is the value that undoes multiplication by another value. In \(\mathrm{GF}(2^8)\), it is a byte that multiplies with another byte to give 0x01.

A2. 0x01.

A3. 0x00.

A4. A byte inverse is not a decimal fraction. It is another byte inside the finite field, and the multiplication uses polynomial multiplication plus AES reduction.

Part C: Identity and zero cases

C1 sample answer. 0x01 is the multiplicative identity, so 0x01 · 0x01 = 0x01. Therefore it is its own inverse.

C2 sample answer. Zero times any byte is zero. Since 0x00 · b can never equal 0x01, zero has no multiplicative inverse.

Part E: Verify 0x02 · 0x8D = 0x01

E1.

\[ \texttt{0x8D}=10001101_2=x^7+x^3+x^2+1 \]

E2.

\[ x(x^7+x^3+x^2+1)=x^8+x^4+x^3+x \]

E3.

\[ x^8\equiv x^4+x^3+x+1 \] \[ x^8+x^4+x^3+x\equiv(x^4+x^3+x+1)+x^4+x^3+x=1 \]

Final result: 0x01.

Part H: AES S-box connection

H1 sample answer. First AES takes the multiplicative inverse in \(\mathrm{GF}(2^8)\), with 0x00 handled specially. Then it applies an affine transformation.

H2 sample answer. The inverse step is nonlinear because it does not behave like a simple XOR, shift, or linear bit-mixing operation. It creates a more complex relationship between input bits and output bits.

H3 sample answer. AES applies an affine transformation after inversion.

Part I: Concept checks

I1 sample answer. An irreducible modulus polynomial creates a field. In a field, every nonzero element has a multiplicative inverse.

I2 sample answer. If \(a\cdot b=1\), then \(b\cdot a=1\) because multiplication in this field is commutative. Therefore each is the inverse of the other.

I3 sample answer. The inverse relationship is defined inside \(\mathrm{GF}(2^8)\), so the product must be computed using finite-field multiplication and AES reduction.
