How to use this workbook

Treat each matrix row as one output-bit recipe. A 1 means include the input bit. A 0 means ignore it. Combine included bits using XOR. Remember: over \(\mathrm{GF}(2)\), \(1+1=0\).

Part H: Linear versus affine

ProblemClassificationReason
H1LinearIt has the form \(y=Av\).
H2AffineIt has a constant vector \(c\) added by XOR.
H3LinearA permutation matrix is still a matrix multiplication with no added constant.
H4AffineIt adds a fixed constant vector after matrix multiplication.

Part I: Invertibility concept checks

I1 sample answer. A matrix is invertible if another matrix exists that undoes its transformation, so applying both returns the original vector.

I2 sample answer. Decryption must recover earlier cipher states. If a cipher layer loses information, it cannot be reversed reliably.

I3 sample answer. No. Some matrices collapse different inputs to the same output, which means information is lost and no inverse exists.

Part J: Cryptography connection

J1 sample answer. The AES S-box affine layer can be written as \(s=Ab\oplus c\), where \(A\) is a fixed binary matrix and \(c\) is the constant from 0x63.

J2 sample answer. Linear layers spread influence and create diffusion, but by themselves they preserve algebraic structure. Ciphers also need nonlinear layers such as S-boxes.
