How to use this workbook

Work slowly through the modular arithmetic. The goal is not to memorize the toy numbers, but to see why Alice and Bob arrive at the same shared secret while an eavesdropper gets stuck at the discrete logarithm problem.

Quick reference

Public values
\[ A=g^a\bmod p \] \[ B=g^b\bmod p \]
Shared secret
\[ S_A=B^a\bmod p \] \[ S_B=A^b\bmod p \] \[ (g^b)^a=(g^a)^b=g^{ab} \]

Alice\(B=g^b\bmod p\)\(S_A=B^a\bmod p=(g^b)^a\bmod p=g^{ab}\bmod p\)Exponent product is \(ab\).

E1 sample answer. In exponentiation, raising a power to another power multiplies the exponents. \((g^b)^a=g^{ba}\) and \((g^a)^b=g^{ab}\). Since \(ab=ba\), the results match.

Part H: Discrete Logarithm Problem

H1 sample answer. Given \(g\), \(p\), and \(A=g^a\bmod p\), the Discrete Logarithm Problem is to find the hidden exponent \(a\).

H2 sample answer. Modular exponentiation mixes the exponent into a wrapped result. In suitable groups, reversing that mixing is hard, like trying to reconstruct ingredients after blending.

Part N: Concept checks

N1 sample answer. Diffie-Hellman is a key exchange method where public parameters \(p\) and \(g\) are used with private exponents \(a\) and \(b\). Alice sends \(A=g^a\bmod p\), Bob sends \(B=g^b\bmod p\), and both compute the same shared secret.

N2 sample answer. Alice computes \((g^b)^a=g^{ba}\), while Bob computes \((g^a)^b=g^{ab}\). Since \(ab=ba\), they arrive at the same value.

N3 sample answer. The Discrete Logarithm Problem asks an attacker to recover the hidden exponent from the public modular exponentiation result. Diffie-Hellman relies on this being hard in properly chosen groups.

N4 sample answer. This tutorial explained the mathematics of classical Diffie-Hellman key exchange. Alice and Bob use public parameters and private exponents to publish public values, then compute the same shared secret using modular exponentiation. The eavesdropper sees the public values but should not be able to recover the private exponents or shared secret because of the Discrete Logarithm Problem. The raw shared secret should feed a KDF, and authentication is still needed against active attackers.
