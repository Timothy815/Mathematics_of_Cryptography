Part 5 True or false

Select True or False for each statement. Think carefully — some of these target the common mistakes section.

Question 20
0x0A equals 10 in decimal.
Correct: True
Question 21
The bytes 00110011 and 110011 represent the same value in a cryptographic context.
Correct: False  (leading zeros encode position; fixed-width matters in cryptography)
Question 22
In an 8-bit system, the result of adding 1 to 11111111 is 100000000.
Correct: False  (the carry bit is lost; the result wraps to 00000000)
Question 23
0xFF and 111111112 represent the same value.
Correct: True  (both equal 255)
Question 24
A single hex digit can represent exactly 16 distinct values (0 through 15).
Correct: True
