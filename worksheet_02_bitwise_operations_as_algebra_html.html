Part 1: Core Definitions

Q1. Define XOR in your own words. What does it output and when?
XOR outputs 1 when two input bits are different, and 0 when they are the same.
Q2. Define AND. What does it output and when?
AND outputs 1 only when both input bits are 1. All other input combinations produce 0.
Q3. What is a bit mask? Give one example of how AND masking is used.
A bit mask is a binary pattern used with AND, OR, or XOR to select, clear, or force specific bit positions. Example: AND with 0x0F (00001111) keeps only the low nibble of a byte.
Q4. Explain the difference between a logical shift and a rotation.
A logical shift moves bits and fills empty positions with 0; bits that fall off the edge are permanently lost. A rotation moves bits and wraps edge bits around to the opposite side, so no bits are lost.

Q29. Reflection.

Why is AND masking dangerous if used as a direct encryption step? Use the idea of reversibility in your answer.

AND masking clears bits to 0, permanently destroying information. If bits are lost during encryption, decryption cannot recover the original value from the result alone. A cipher step must be reversible so the recipient can decrypt — AND masking violates that requirement.
