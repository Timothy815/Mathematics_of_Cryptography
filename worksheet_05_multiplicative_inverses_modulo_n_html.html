Definition\(a^{-1} \pmod{n}\) is the unique \(x \in \{0,\ldots,n-1\}\) such that \(a \cdot x \equiv 1 \pmod{n}\).
Existence\(a^{-1} \pmod{n}\) exists if and only if \(\gcd(a,n)=1\).
VerifyTo check: multiply \(a \cdot x\) and reduce mod \(n\). If the result is 1, \(x\) is the inverse.
Affine Decrypt\(P \equiv a^{-1}(C - b) \pmod{26}\). Add 26 if result is negative.

The affine cipher encrypts as \(C \equiv aP + b \pmod{26}\). To decrypt: \(P \equiv a^{-1}(C - b) \pmod{26}\). Add 26 if the intermediate value is negative.
