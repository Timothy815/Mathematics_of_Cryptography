Quick Reference — tap to expand
xtime rule: Top bit = 0 → \(\operatorname{xtime}(a)=a\ll1\). Top bit = 1 → \(\operatorname{xtime}(a)=(a\ll1)\oplus\texttt{0x1B}\).
Multiply by 0x03: \(a\cdot\texttt{0x03}=\operatorname{xtime}(a)\oplus a\).
General multiply: Build xtime chain \(a,\,xa,\,x^2a,\ldots\). For each bit \(i\) set in \(b\), XOR in \(x^i a\).
AES modulus: \(m(x)=x^8+x^4+x^3+x+1\) (0x11B); folded part = 0x1B.

Part D — xtime chain for 0x57

\(x^0 a=\texttt{0x57}\), \(x^1 a=\texttt{0xAE}\), \(x^2 a=\texttt{0x47}\), \(x^3 a=\texttt{0x8E}\), \(x^4 a=\texttt{0x07}\)

Note: \(\operatorname{xtime}(\texttt{0xAE})\): top bit 1, shift → \(\texttt{0x5C}\), XOR \(\texttt{0x1B}\) → \(\texttt{0x47}\). \(\operatorname{xtime}(\texttt{0x47})\): top bit 0, shift → \(\texttt{0x8E}\). \(\operatorname{xtime}(\texttt{0x8E})\): top bit 1, shift → \(\texttt{0x1C}\), XOR \(\texttt{0x1B}\) → \(\texttt{0x07}\).

Part E — Full products

E1. \(\texttt{0x13}=x^4+x+1\). Select \(x^4 a\oplus x^1 a\oplus x^0 a=\texttt{0x07}\oplus\texttt{0xAE}\oplus\texttt{0x57}=\texttt{0xFE}\)

E2. \(\texttt{0x0B}=x^3+x+1\). Select \(x^3 a\oplus x^1 a\oplus x^0 a=\texttt{0x8E}\oplus\texttt{0xAE}\oplus\texttt{0x57}=\texttt{0x77}\)

Check E2: \(\texttt{0x8E}\oplus\texttt{0xAE}=\texttt{0x20}\); \(\texttt{0x20}\oplus\texttt{0x57}=\texttt{0x77}\). ✓

Part F — Conceptual questions

F1 sample. Multiplying by \(x\) shifts every term one degree higher. When the original byte contains an \(x^7\) term, the shift produces an \(x^8\) term. The AES modulus says \(x^8\equiv x^4+x^3+x+1\), so we fold it by XORing with \(\texttt{0x1B}=x^4+x^3+x+1\). Without this fold the result would exceed the 8-bit byte range.

F2 sample. \(\texttt{0x11B}\) (nine bits) is the full AES irreducible modulus polynomial \(x^8+x^4+x^3+x+1\). \(\texttt{0x1B}\) (eight bits) is just the lower portion \(x^4+x^3+x+1\) — the term that replaces \(x^8\) after folding.

F3 sample. "Closed" means that multiplying any two bytes in \(\mathrm{GF}(2^8)\) always produces another byte (a value between 0x00 and 0xFF). The AES modulus reduction guarantees this: whenever the product exceeds degree 7, it is folded back inside the byte range.
